Server name adalah label unik yang dberikan kepada sebuah perangkat dalam suatu jaringan, jika diibaratkan 
IP Address adalah nomor KTP sedangkan server name adalah nama panggilan orang tersebut agar mudah diingat

# Server Name Pada Level Sistem 
Ini adalah identitas komputer anda didalam jaringan lokal. Saat anda membuka terminal di Ubuntu, nama yang muncul setelah tanda @ adalah Server
Name/Hostname sistem anda.

# Server Name Pada Web Server 
Jika anda menggunakan web server seperti Nginx atau Apache, server_name adalah direktif yang menentukan domain mana yang akan dilayani oleh blok
konfigurasi tersebut. satu server dengan satu IP bisa menampung banyak website. Web server menggunakan Server Name yang dikirim oleh browser untuk 
menentukan website mana yang harus ditampilkan

konfigurasi berikut menggunakan nginx
```bash 
	listen 80;
	server_name www.bisnis.com;
	root /var/www/bisnis; ```


# Server dalam local DNS Mapping
Dalam ``` /etc/host ``` server name adalah alias yang anda buat untuk sebuah IP. Misalnya anda punya server database dengan IP 10.0.0.5. daripada 
menghapal angka tersebut anda memberinya server name

```bash 
	10.0.0.5   db-produksi ```
sekarang db-produksi adalah server name yang valid untuk komputer anda
