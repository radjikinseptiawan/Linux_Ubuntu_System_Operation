# Local DNS Mapping
LDNSM adalah proses menghubungkan sebuah nama domain (hostname) ke alamat IP tertentu secara manual di dalam siste komputer lokal,tanpa harus bertanya
ke server DNS publik di internet

# Bagaimana Cara Kerjanya?
Setiap sistem operasi memiliki file khusus yang disebut file hosts. disinilah pemetaan manual ini terjadi. urutan kerjanya sebagai
berikut:
1. Anda mengetik myserver.tst di browser
2. Sistem mengecek file hosts lokal
3. Jika ditemukan pemetaan, komputer langsung membuka IP tersebut
4. Jika tidak ada di file lokal, barulah sistem bertanya ke DNS Server di Internet
Lokasi file mapping di Linux Ubuntu ada di 
``` /etc/hosts ```
